# dockerfiles
本仓库不定期收集、整理、编写常用优秀软件和服务的 Dockerfile 文件，分享成熟、稳定、好用的 docker-compose.yml 部署配置文件。

当前已全部构建 X86 和 ARM 架构 docker 镜像，可以直接访问 [Docker Hub](https://hub.docker.com/u/stilleshan) 官方镜像仓库直接使用。
